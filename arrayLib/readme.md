## A functional library for working with arrays.

Functionality (arrayLib):

- take(arr,n) - lodash;
- skip(arr, n) - lodash;
- map(arr, cb) - lodash;
- reduce(arr, cb, init) - lodash;
- filter(arr, cb) - lodash;
- foreach(arr, cb) - lodash.


Functionality (arrayLib2):

- sort(bubble, insertion);
- min(arr);
- max(arr);
- avg(arr);
- getMaxSubSum(arr).


Tested with Jest.
